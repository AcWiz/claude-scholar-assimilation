# ACL 投稿清单

## 格式要求
- [ ] Long Paper <= 8 页 / Short Paper <= 4 页
- [ ] 使用 ACL style files
- [ ] 双栏格式
- [ ] 引用不计入页数

## 必选内容
- [ ] Abstract: 150-200 words
- [ ] 方法描述完整
- [ ] 在标准 NLP benchmark 上评估
- [ ] **Limitations 章节（强制）**
- [ ] **Ethics Statement（如适用，强制）**
  - [ ] 人类受试者 IRB 批准
  - [ ] 数据隐私和同意
  - [ ] 计算资源环境影响

## 匿名化
- [ ] 双盲审查
- [ ] 移除所有身份信息
- [ ] 自引用匿名化

## 统计与实验
- [ ] NLP 任务特定 baseline
- [ ] 人工评估（生成类任务必须）
- [ ] 错误分析
- [ ] 多语言验证（如适用）
- [ ] 数据标注质量说明（inter-annotator agreement）

## 数据与伦理
- [ ] 数据集偏见分析
- [ ] 标注协议说明
- [ ] Reproducibility checklist 填写
- [ ] 数据使用许可说明

## 补充材料
- [ ] 附录格式一致
- [ ] 额外实验和分析
- [ ] 代码可用性说明
