# ICML 投稿清单

## 格式要求
- [ ] 主论文 <= 8 页（Camera-Ready 9 页）
- [ ] 使用官方 LaTeX 模板
- [ ] 双栏格式
- [ ] 引用不计入页数

## 必选内容
- [ ] Abstract 清晰总结贡献
- [ ] 方法可复现
- [ ] 全面的实验评估
- [ ] **Broader Impact Statement（强制，<= 1 页）**
- [ ] 讨论正面和负面社会影响
- [ ] 考虑偏见、公平性、环境影响

## 匿名化
- [ ] 双盲审查
- [ ] 移除所有作者身份信息
- [ ] 自引用匿名化
- [ ] 代码仓库匿名化

## 统计与实验
- [ ] 理论基础完整
- [ ] 实验对比公平（相同计算预算）
- [ ] 算法复杂度分析
- [ ] 收敛性分析（如适用）
- [ ] 超参数敏感性分析

## 补充材料
- [ ] 附录格式与主论文一致
- [ ] 证明和推导完整
- [ ] 额外实验结果
- [ ] 代码可用性说明
