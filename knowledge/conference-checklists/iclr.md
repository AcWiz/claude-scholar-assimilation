# ICLR 投稿清单

## 格式要求
- [ ] 主论文 <= 9 页（Camera-Ready 10 页）
- [ ] 使用官方 LaTeX 模板
- [ ] 双栏格式
- [ ] 引用不计入页数

## 必选内容
- [ ] Abstract 总结贡献
- [ ] 方法可复现
- [ ] 全面的实验评估
- [ ] **Limitations 章节（强制）**
- [ ] **LLM Disclosure（如使用 LLM，强制）**
  - [ ] 描述 LLM 使用方式
  - [ ] 模型细节
  - [ ] 局限性说明
  - [ ] 自动文本生成标注

## 匿名化
- [ ] 双盲审查
- [ ] 移除所有身份信息
- [ ] 注意 ICLR 采用公开审稿

## 统计与实验
- [ ] 与最新方法公平对比
- [ ] 统计检验
- [ ] 消融实验充分
- [ ] 可视化分析（t-SNE, attention 等）
- [ ] 强烈鼓励提供代码

## 补充材料
- [ ] 附录清晰组织
- [ ] 主论文自包含
- [ ] 代码链接（匿名化）
- [ ] 数据获取说明
