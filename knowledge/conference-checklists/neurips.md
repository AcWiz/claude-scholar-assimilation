# NeurIPS 投稿清单

## 格式要求
- [ ] 主论文 <= 9 页（不含引用）
- [ ] 使用官方 LaTeX 模板
- [ ] 9pt 字体，双栏格式
- [ ] 1 inch 页边距

## 必选内容
- [ ] Abstract: 150-250 words
- [ ] 贡献清晰陈述
- [ ] 方法足够详细以复现
- [ ] 全面的实验评估
- [ ] 讨论限制
- [ ] 16 项 Submission Checklist 全部填写

## 匿名化
- [ ] 移除作者名和机构
- [ ] 自引用匿名化
- [ ] GitHub 链接匿名化
- [ ] 致谢已移除
- [ ] 补充材料也已匿名化

## 统计与实验
- [ ] 报告误差（SD/SE，明确标注）
- [ ] 统计显著性检验
- [ ] 消融实验支撑核心声明
- [ ] 包含最新 SOTA baseline

## 补充材料
- [ ] 附录组织清晰（S1, S2...）
- [ ] 主论文自包含
- [ ] 代码/数据可用性声明
- [ ] 录用后需准备 Lay Summary（1 页）
