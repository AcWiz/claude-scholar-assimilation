# CVPR 投稿清单

## 格式要求
- [ ] 主论文 <= 8 页（Camera-Ready +1 页）
- [ ] 使用官方 LaTeX 模板
- [ ] 双栏格式
- [ ] 引用不计入页数

## 必选内容
- [ ] Abstract 清晰总结
- [ ] 方法描述可复现
- [ ] 全面的视觉实验评估
- [ ] 定量结果 + 定性结果（可视化比较）

## 匿名化
- [ ] 双盲审查
- [ ] 移除所有身份信息
- [ ] 代码仓库匿名化

## 统计与实验
- [ ] 标准视觉 benchmark 评估
- [ ] 最新 SOTA baseline 对比
- [ ] **可视化比较图（qualitative results）**
- [ ] **计算效率分析（FPS、推理时间、参数量）**
- [ ] 消融实验（每个模块、每个 loss）
- [ ] 失败案例分析
- [ ] 参数量和 FLOPs 对比表

## 图表要求
- [ ] 专业架构图
- [ ] 可视化比较图高质量
- [ ] 图表矢量图格式
- [ ] 字体清晰可读

## 补充材料
- [ ] 独立 PDF 补充
- [ ] 视频演示（如适用，重要加分项）
- [ ] 更多可视化结果
- [ ] 代码可用性说明
- [ ] 跨数据集泛化实验
