# /poster - Academic Poster Designer

Design an academic poster for your conference presentation.

## What This Command Does

1. Guides you through poster design process
2. Provides layout templates and design guidelines
3. Helps with visual hierarchy and content organization

## Workflow

1. Ask about poster size requirements (default: A0)
2. Read the paper to extract key content
3. Suggest layout structure:
   - 3 columns: cleaner, better for method-focused papers
   - 4 columns: more content, better for results-heavy papers
4. Organize content sections:
   - Title bar (title, authors, affiliations, logos)
   - Introduction / Motivation
   - Method (with key figure)
   - Results (key tables/figures)
   - Conclusion
   - References (compact)
   - QR code for paper/code
5. Provide design guidelines:
   - Visual hierarchy: title > section headers > body
   - Color scheme matching paper/institution
   - Font sizes: title 80pt, headers 48pt, body 28pt
   - High-resolution images (300 DPI)

## Output

Generate:
- Poster content layout with section placement
- Text content for each section (concise, poster-optimized)
- Figure placement recommendations
- Color scheme suggestions

## Tips

- Start designing 1-2 weeks before printing
- Print a test version before final printing
- Keep text concise and readable from 1 meter away
