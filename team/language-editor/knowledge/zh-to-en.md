# 中转英翻译参考

> 来源：awesome-ai-research-writing prompt 01

## 核心规则

### 1. 视觉与排版
- 尽量不使用加粗、斜体或引号
- 保持 LaTeX 源码纯净，不添加无意义的格式修饰

### 2. 风格与逻辑
- 逻辑严谨，用词准确，表达凝练连贯
- 尽量使用常见单词，避免生僻词
- 尽量不使用破折号（--），推荐从句或同位语替代
- 拒绝使用 \item 列表，必须使用连贯段落
- 去除"AI 味"，行文自然流畅

### 3. 时态规范
- 统一使用一般现在时描述方法、架构和实验结论
- 仅在明确提及特定历史事件时使用过去时

### 4. 特殊字符转义
- `95%` -> `95\%`
- `model_v1` -> `model\_v1`
- `R&D` -> `R\&D`
- 保持数学公式原样（保留 `$` 符号）

### 5. 输出格式
- Part 1 [LaTeX]：全英文翻译结果
- Part 2 [Translation]：对应中文直译（核对逻辑）

### 6. 自查协议
- 审稿人视角：检查过度排版、逻辑跳跃、未翻译中文
- 立即纠正发现的问题

## 翻译注意事项

### 学术术语处理
- 首次出现给出完整英文术语
- 后续使用缩写
- 专有名词保持英文（NeurIPS, Transformer, BERT）

### 句式转换
- 中文的"主题+评论"结构 -> 英文的 SVO 结构
- 避免中文式长定语 -> 使用关系从句或分句
- 中文的隐含逻辑 -> 英文需要明确的逻辑连接词

### 常见翻译对照
| 中文 | 英文 |
|------|------|
| 本文提出 | We propose |
| 实验表明 | Experiments show that |
| 与...相比 | Compared to / Unlike |
| 进一步地 | Furthermore / Moreover |
| 具体来说 | Specifically / In particular |
| 值得注意的是 | Notably |
| 为了解决这个问题 | To address this issue |
