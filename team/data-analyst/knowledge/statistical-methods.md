# ML/AI 实验统计分析方法

> 来源：claude-scholar results-analysis 知识库

## 基础统计量

| 统计量 | 公式 | 报告格式 | 用途 |
|--------|------|----------|------|
| 均值 | mu = (Sigma x) / n | "准确率 85.3%" | 描述集中趋势 |
| 标准差 (SD) | SD = sqrt[Sigma(x-mu)^2 / (n-1)] | "85.3% +- 2.1% (SD)" | 描述数据分散程度 |
| 标准误 (SE) | SE = SD / sqrt(n) | "85.3% +- 0.7% (SE)" | 估计均值不确定性 |
| 置信区间 | CI = mu +- t * SE | "85.3% [95% CI: 83.9%, 86.7%]" | 参数估计范围 |

**重要**：论文中必须明确说明使用的是标准差还是标准误。

## 预先检验（必须执行）

### 1. 正态性检验
- **Shapiro-Wilk**: 样本量 n < 50, p > 0.05 -> 接受正态
- **Kolmogorov-Smirnov**: 样本量 n >= 50
- **Q-Q 图**: 图形化辅助判断

**不满足正态性**：数据转换(log, sqrt) / 非参数检验 / 增加样本量

### 2. 方差齐性检验
- **Levene 检验**: 默认选择，对非正态鲁棒
- **Bartlett 检验**: 数据严格正态时

**不满足方差齐性**：Welch's t-test / Welch's ANOVA

## 统计检验选择流程

```
数据类型 -> 配对/独立？ -> 组数？ -> 正态性？ -> 方差齐性？

配对数据：
  两组 + 正态 = 配对 t-test
  两组 + 非正态 = Wilcoxon 符号秩检验
  多组 + 正态 = 重复测量 ANOVA
  多组 + 非正态 = Friedman 检验

独立数据：
  两组 + 正态 + 齐方差 = 独立 t-test
  两组 + 正态 + 不齐方差 = Welch's t-test
  两组 + 非正态 = Mann-Whitney U 检验
  多组 + 正态 + 齐方差 = ANOVA + 事后检验
  多组 + 正态 + 不齐方差 = Welch's ANOVA + Games-Howell
  多组 + 非正态 = Kruskal-Wallis + Dunn 检验
```

## 参数检验

### t-test（两组对比）
- **独立样本 t-test**: "方法 A (85.3% +- 2.1%) 显著优于方法 B (82.1% +- 1.8%), t(18) = 3.45, p = 0.003"
- **配对样本 t-test**: "在 10 个数据集上，方法 A 显著优于方法 B, t(9) = 4.23, p = 0.002"
- **Welch's t-test**: 方差不齐时使用

### ANOVA（多组对比）
- **单因素 ANOVA**: "不同方法之间存在显著差异, F(3, 36) = 8.45, p < 0.001"
- **重复测量 ANOVA**: 同一对象多条件
- **双因素 ANOVA**: 两个因素及交互作用

### 事后检验
| 方法 | 使用场景 | 保守程度 |
|------|----------|----------|
| Tukey HSD | 默认选择 | 适中 |
| Bonferroni | 严格控制 Type I 错误 | 保守 |
| Dunnett | 与单一对照组比较 | 专用 |
| Scheffe | 复杂对比 | 最保守 |

## 非参数检验

| 检验方法 | 替代 | 使用场景 |
|----------|------|----------|
| Wilcoxon 符号秩 | 配对 t-test | 配对数据，两组，非正态 |
| Mann-Whitney U | 独立 t-test | 独立数据，两组，非正态 |
| Kruskal-Wallis | ANOVA | 独立数据，多组，非正态 |
| Friedman | 重复测量 ANOVA | 配对数据，多组，非正态 |
| Sign 检验 | 配对 t-test | 只关心胜负方向 |

## 多重比较校正

| 方法 | 保守程度 | 使用场景 |
|------|----------|----------|
| Bonferroni (alpha/k) | 最保守 | 少量检验 |
| Holm-Bonferroni | 较保守 | 中等数量检验 |
| FDR (Benjamini-Hochberg) | 较宽松 | 大量检验，探索性分析 |

## 效应量

| 效应量 | 适用场景 | 解释 |
|--------|----------|------|
| Cohen's d | t-test | <0.2 小, 0.2-0.5 中, >=0.8 大 |
| eta^2 | ANOVA | 0.01 小, 0.06 中, 0.14 大 |
| r | 非参数检验 | 0.1 小, 0.3 中, 0.5 大 |

## 实验设计要点

- **重复次数**: 最少 3-5 次，推荐 5-10 次，高方差任务 10+ 次
- **随机种子**: 报告所有使用的随机种子
- **交叉验证**: k-fold (k=5 或 10)

## 报告检查清单

- [ ] 报告均值和标准差/标准误（明确标注）
- [ ] 报告实验重复次数
- [ ] 执行预先检验（正态性、方差齐性）
- [ ] 选择适当的统计检验
- [ ] 报告完整统计信息（检验统计量、自由度、p-value）
- [ ] 报告效应量
- [ ] 多重比较进行校正
- [ ] 说明随机种子设置

## 常见错误

1. Cherry-picking：只报告最好结果 -> 报告所有实验
2. p-hacking：多次分析找显著 -> 预先确定方法
3. 混淆 SD 和 SE -> 明确标注
4. 忽略多重比较 -> 使用 Bonferroni/FDR
5. 只报告 p-value -> 同时报告效应量
