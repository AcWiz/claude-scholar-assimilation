# 顶会投稿要求

> 来源：claude-scholar ml-paper-writing 知识库

## NeurIPS

| 项目 | 要求 |
|------|------|
| 主论文 | 9 页（不含引用） |
| 引用 | 不限 |
| 附录 | 允许，审稿人不必阅读 |
| 审查方式 | 双盲 |
| 模板 | LaTeX，9pt 字体，双栏 |
| 必选章节 | Abstract (150-250 words), Checklist (16 项) |
| 录用后 | Lay Summary (1 页，非技术) |

**Checklist 要点**：匿名化、图表清晰、代码/数据可用性声明、统计显著性、限制讨论、Broader Impact。

## ICML

| 项目 | 要求 |
|------|------|
| 主论文 | 8 页 |
| Camera-Ready | 9 页 |
| 引用 | 不限 |
| 审查方式 | 双盲 |
| 必选章节 | Broader Impact Statement (1 页以内) |

**Broader Impact**：讨论正面和负面社会影响、偏见、公平性、环境影响。

## ICLR

| 项目 | 要求 |
|------|------|
| 主论文 | 9 页 |
| Camera-Ready | 10 页 |
| 引用 | 不限 |
| 审查方式 | 双盲 |
| 必选章节 | Limitations (强制), LLM Disclosure (如使用 LLM) |

**LLM Disclosure**：描述 LLM 使用方式、模型细节、局限性、自动生成文本说明。

## ACL

| 项目 | 要求 |
|------|------|
| Long Paper | 8 页 |
| Short Paper | 4 页 |
| 引用 | 不限 |
| 审查方式 | 双盲 |
| 模板 | ACL style files |
| 必选章节 | Limitations (强制), Ethics Statement (如适用) |

**Ethics**：人类受试者需 IRB 批准、数据隐私、计算资源环境影响。

## CVPR

| 项目 | 要求 |
|------|------|
| 主论文 | 8 页 |
| Camera-Ready | +1 页 |
| 引用 | 不限 |
| 审查方式 | 双盲 |
| 补充材料 | 允许，独立 PDF |

## 通用双盲要求

- 移除作者名和机构
- 匿名化自引用 [Anonymous, 2024]
- 移除暴露身份的致谢
- 避免可识别身份的措辞
- 补充材料也需匿名化

**常见错误**：
- 忘记匿名化 GitHub 链接
- 文件路径中包含作者名
- 致谢中感谢特定同事
- 自引用未匿名化

## 代码和数据可用性

- NeurIPS: 鼓励代码/数据可用性声明
- ICML: 鼓励可复现性
- ICLR: 推荐代码分享

**最佳实践**：
- 匿名化代码仓库链接
- 提供数据获取说明
- 描述超参数和设置
- 说明专有性限制

## 补充材料规范

- 附录允许但非必读
- 用于额外实验、证明、表格
- 主论文保持自包含
- 正文中引用补充材料
- 格式与主论文一致
- 章节编号使用 S1, S2 等

## 投稿前通用清单

**内容**：
- [ ] 创新贡献清晰陈述
- [ ] 相关工作全面
- [ ] 方法可复现
- [ ] 结果支持所有声明
- [ ] 限制已确认
- [ ] Broader Impact（如需要）

**格式**：
- [ ] 页数限制
- [ ] 样式文件正确
- [ ] 引用完整一致
- [ ] 图表清晰可读
- [ ] 公式编号和引用

**匿名**：
- [ ] 作者名移除
- [ ] 致谢匿名化
- [ ] 自引用匿名化
- [ ] GitHub 链接匿名化
