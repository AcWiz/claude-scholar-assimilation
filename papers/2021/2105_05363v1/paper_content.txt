6 Discussion
This paper proposes a new particle п¬Ғltering algorithm, Lang evinized EnKF, by reformulating the EnKF
under the framework of Langevin dynamics. The Langevinized EnKF converges to the right п¬Ғltering
distribution in Wasserstein distance under the big data sce nario that the dynamic systems consists of a
large number of stages and has a large number of observations at each stage. The Langevinized EnKF
inherits theforecast-analysis procedurefrom theEnKFand theuseof mini-batch datafrom thestochastic
gradient MCMC algorithms, enabling it scalable with respec t to both the dimension and the sample size.
The Langevinized EnKF can be applied to both the inverse and d ata assimilation problems.
Like theEnKF, theLangevinized EnKFis developed underthea ssumption that thestate space model
contains no unknown parameters. Extension of the algorithm to the case with unknown parameters can
be done in diп¬Җerent ways. One way is with the state augmentatio n approach. Taking the model (5) as an
example: If the function ПҖ(В·) or more general, the state propagator, contains some unkno wn parameters,
we can augment the state xtto include these parameters. Another way is with the EM algor ithm
(Dempster et al., 1977). Since the Langevinized EnKF is able to sample from the п¬Ғltering distribution for
given parameters, the EM algorithm can be conveniently used for estimating parameters. A related work
is Aicher et al. (2019), however, where the п¬Ғltering distrib ution is simulated using traditional sequential
Monte Carlo. Third, the parameters can also be estimated usi ng an adaptive SGMCMC algorithm.
